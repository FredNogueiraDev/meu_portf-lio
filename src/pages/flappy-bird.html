Sobre o Flappy Bird

Desenvolvi este projeto para exercitar meus conhecimentos em HTML, CSS e JavaScript com a ajuda do professor Leonardo Leitão no curso de programação para web da Cod3r. O objetivo do projeto foi implementar um jogo Flappy Bird utilizando manipulação do DOM, em que o jogador controla um pássaro que deve passar entre canos para marcar pontos e bater recordes.
