Sobre o Get a Pet

Este projeto foi desenvolvido com o objetivo de exercitar meus conhecimentos em HTML, CSS, JavaScript, MongoDB e NodeJS. Para isso, foi implementado um site de adoção de animais de estimação.

O objetivo é ajudar as pessoas interessadas em doar animais a encontrar pessoas que desejam adotá-los. Para facilitar esse processo, os animais disponíveis para adoção são exibidos na página inicial do site, permitindo que todos possam visualizá-los.

O projeto, intitulado "Get a Pet", foi construído seguindo a arquitetura MVC e aplicando boas práticas de programação, incluindo testes e validações. Isso garante a robustez do sistema baseado em Node.

O projeto foi realizado com o apoio do professor Matheus Battisti, no curso "NodeJS do zero a Maestria".

Como utilizar o projeto

Para utilizar o Get a Pet, você deve ter os seguintes softwares instalados em sua máquina:

  • NodeJS v19.4.0
  • NPM v9.2.0
  • MongoDB v5.0.14
